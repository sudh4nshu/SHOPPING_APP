I wrote a simple Shopping Cart program to demonstrate how Object-Oriented Programming (OOP) works in Python.

This script breaks down the three essential types of methods every Python developer should know:

🛒 Instance Methods (add_product, remove_product): Used to manage actions unique to each customer's specific cart.

🌐 Class Methods (@classmethod): Used to manage data shared globally by the entire class (e.g., updating the store name from "Amazon" to "Flipkart").

📋 Static Methods (@staticmethod): Used for independent utility functions that don't need any cart or class data (like displaying general shopping rules).
